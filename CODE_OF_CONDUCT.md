# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## Summary

We expect everyone participating in this project — issues, pull requests, discussions, code review, community channels — to be respectful, constructive, and patient with one another. Focus on the work, not the person.

## What we ask of you

- Use welcoming and inclusive language
- Respect differing viewpoints and experiences
- Accept constructive feedback gracefully
- Focus on what is best for the community
- Show empathy toward other contributors

## What we ask you not to do

- Personal attacks, insults, or derogatory comments
- Harassment in any form
- Publishing others' private information without explicit permission
- Other conduct that a reasonable person would find inappropriate in a professional setting

## Reporting

If you experience or witness behavior that violates this code, please contact the maintainers privately. Reports will be handled confidentially. The full Contributor Covenant text describes the enforcement process.

## Scope

This applies to all project spaces — repository, issue tracker, pull requests, discussions, and any official channels — and to public spaces when an individual is representing the project.
