# Security Policy

Thank you for helping keep Skill Wiki users safe.

## Reporting a vulnerability

Please report security issues privately. Do **not** open a public GitHub issue.

- Open a [GitHub security advisory](https://github.com/skill-wiki/skill-wiki.github.io/security/advisories/new) — preferred channel.
- Or email **security@skill-wiki** *(placeholder — to be replaced with the real address before v1.0)*.

Please include:

1. A clear description of the issue
2. Steps to reproduce, or a minimal proof-of-concept
3. The version or commit you tested
4. Your assessment of impact

## What to expect

- We aim to acknowledge reports within **7 days**.
- We aim to publish a fix and advisory within **90 days** of acknowledgement.
- We will credit the reporter in the advisory unless you ask us not to.

## Supported versions

Only the **current major version** of the website receives security fixes.
